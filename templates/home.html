This page documents how to use the DNS Resolver tool.

Resolver Page

Enter a domain and choose a record type (A, AAAA, MX, etc.). Results from different DNS servers will be grouped by identical output. Includes latency, TTL, flags, and DNSSEC status.

Trace Page

Shows the path a DNS query takes from a root server down to the authoritative server. Each step includes answer, authority, and additional sections.

Server Summary

Highlights which DNS servers responded, what they returned, and how fast they were.
